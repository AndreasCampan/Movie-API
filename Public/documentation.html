Objective:

To build the server-side component of a movies web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

Return data (description, genre, director) about a single movie by title GET /movies/:title title none

A JSON response with the requested movie title as an object

e.g.
{
  "title": "Forrest Gump"
}

Return data about a genre description by name GET /movies/genres/:name name none

An html response with a description of the requested genre

e.g.
Science fiction (sci-fi) - a film genre that uses speculative, fictional science-based depictions of phenomena that are not fully accepted by mainstream science, such as extraterrestrial lifeforms, alien worlds, extrasensory perception and time travel, along with futuristic elements such as spacecraft, robots, cyborgs, interstellar travel or other technologies...

Return data about a director (bio, birth/death year) by name GET /directors/:name name none

An html response with a description of the requested director

e.g.
Ridley Scott - an English film director and producer. He has directed the science fiction horror film Alien, the neo-noir dystopian film Blade Runner, the road adventure film Thelma & Louise...
